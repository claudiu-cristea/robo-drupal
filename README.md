# Robo Drupal Tasks

Provides useful Robo commands for Drupal projects.

## Install

Run:

```bash
$ composer require --dev claudiu-cristea/robo-drupal-tasks
```

or add this line to your project `composer.json`:

```json
    "require": {
        "claudiu-cristea/robo-drupal-tasks": "dev-master"
    },
```
